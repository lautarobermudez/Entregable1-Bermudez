🏪 Simulador de Tienda Virtual

📝 Descripción del Proyecto

Este simulador de tienda virtual es una aplicación interactiva desarrollada en JavaScript que permite gestionar un inventario de productos y procesar ventas de manera dinámica. El sistema está diseñado para demostrar el uso de estructuras fundamentales de programación como variables, funciones, arrays, objetos y estructuras de control.

La aplicación simula el funcionamiento de una tienda real donde el usuario puede cargar productos al inventario, realizar ventas con cálculos automáticos de precios (incluyendo IVA y descuentos), gestionar el stock y generar reportes de ventas. Todo el sistema funciona a través de la consola del navegador y cuadros de diálogo interactivos.

🎯 Funcionalidades Implementadas

  • Gestión dinámica de inventario con arrays y objetos
  • Sistema de ventas con cálculo automático de totales
  • Aplicación de IVA y descuentos por cantidad
  • Control de stock en tiempo real
  • Generación de reportes de ventas y productos más vendidos
  • Interfaz interactiva usando Prompt, Confirm y Alert
  • Validación de datos de entrada
  • Menú cíclico para múltiples operaciones

🚀 Instrucciones de Uso

1. Abre las herramientas de desarrollador (F12)

2. Ve a la pestaña "Console"

3. El simulador se iniciará automáticamente

4. Sigue las instrucciones en los cuadros de diálogo

5. Observa los resultados en la consola

📊 Presiona F12 y ve a Console para usar el simulador
